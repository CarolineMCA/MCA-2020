Week 5: Standards in curation

Task 1

Creating 1000-song database - metadata

The metadata schema listed below represents the information I believe would be important in describing a music database with 1000 songs. Such variety of metadata would be able to describe many different pieces of music. This information would not only provide data about the original file such as artist name, album title, song title, but would also provide inforamtion about: encoding and transcription of the files, any changes that have been made, a larger piece of work that the file is part of, and would, e.g. allow to track the progress of and subsequent work on the file, by providing version information.

'After the Gold Rush' metadata file can be accessed here.

Metadata about the original song - outline

title title of an original song
album title name of an album on which the song appeared first
original release date date when the song was released
composer name/s of the creator of the original song
lyricist person/s who wrote the text of a song
producer name/s of the people involved in production fo the song
lable name of the lable under which the song was published
genre name of a genre song can be classified as

Metadata about the file in my repository - outline

title title of a transcired piece
composer name/s of the creator of the intellectual content of a musical work
lyricist person/s who wrote the text of a song
arranger person/s who transcribes a musical composition
encoder person/s who encodes a musical composition
publisher names of entites responsible for publication of the file
publisher location geographic location of the publisher
date of publication date of publication of the file
source of publication source of the original file
size of file size of the file
copyright/use restrictions copyright restrictions
purpose a purpose of the transcription of the file
encoding changes description of any changes that occured during the encoding process
series information on the larger piece of work that contains the fle
edition which version/edition of the file it is
